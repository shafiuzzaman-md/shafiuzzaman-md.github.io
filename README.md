# Md Shafiuzzaman — Faculty Website

Vite + React + Tailwind site optimized for GitHub Pages.

## Quick start
```bash
npm ci
npm run dev
```

## Deploy (GitHub Pages)
- This repo includes a GitHub Actions workflow at `.github/workflows/pages.yml`.
- Push to `main` and GitHub Pages will auto-deploy.
- If you use a custom domain, add it in Settings → Pages and commit a `CNAME` file in the repo root.
